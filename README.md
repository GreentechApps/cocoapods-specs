# Greentech Apps CocoaPods Specs

Private CocoaPods specs repository for Greentech Apps Foundation.

## Usage

Add this repo to your CocoaPods installation:
```bash
pod repo add greentech-specs https://github.com/GreentechApps/cocoapods-specs.git
```

Then in your Podfile:
```ruby
source 'https://github.com/GreentechApps/cocoapods-specs.git'
source 'https://cdn.cocoapods.org/'
```
